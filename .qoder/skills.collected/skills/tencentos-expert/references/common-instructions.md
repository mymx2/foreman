# 共享 Prompt 片段：通用指令

## 安全注意事项

在执行任何系统命令之前，请注意：

1. **确认操作环境**：确保在正确的服务器上执行
2. **备份重要数据**：执行修改操作前先备份
3. **了解命令影响**：清楚每个命令的作用和副作用
4. **记录操作日志**：保留操作记录以便回溯

## 通用信息收集

开始诊断前，建议先收集以下基本信息：

```bash
# 系统基本信息
uname -a
cat /etc/os-release

# 系统运行时间和负载
uptime

# 内存使用情况
free -h

# 磁盘使用情况
df -h

# CPU 信息
lscpu | head -20
```

## 输出格式规范

```

发现的问题:
1. 问题描述 1
2. 问题描述 2

建议操作:
1. 建议 1
2. 建议 2

详细信息:
...
```

### 命令执行输出

```
> 执行命令: <command>
> 执行时间: <timestamp>
> 执行结果: 成功/失败

输出内容:
...
```
